# Benchmarks

Benchmark results are generated by running the Jupyter notebooks on Google Colab Pro and auto-saved here.

## Structure

```
results/
├── elementwise/   # vector_add, activations, fused_elementwise
├── reductions/    # vector_sum, max_min, softmax, layer_norm
├── scanning/      # prefix_sum, cummax
├── matmul/        # naive_matmul, tiled_matmul, batched_matmul
├── fft/           # fft_kernel
└── attention/     # naive_attention, sdpa, flash_v1, flash_v2
```

Each kernel produces:
- `<kernel_name>_benchmark.csv` — raw timing data (size, Triton ms, PyTorch ms, GB/s or TFLOPS)
- `<kernel_name>_benchmark.png` — performance plot

## Metrics

| Category | Metric |
|---|---|
| Elementwise | GB/s |
| Reductions | GB/s |
| Softmax / LayerNorm | GB/s |
| Matmul | TFLOPS |
| Attention | TFLOPS |
| FFT | GFLOPS |

## Benchmark Settings

All benchmarks use `triton.testing.do_bench(warmup=25, rep=100)`.
